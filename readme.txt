
--------------------------------------------
V-WSP variable reduction for MATLAB - version 1
--------------------------------------------
The V-WSP variable reduction for MATLAB is a MATLAB routine for calculating unsupervised variable reduction based on V-WSP algorithm.

This is the version 1.0 of WSP variable reduction for MATLAB

V-WSP variable reduction for MATLAB has been released by Milano Chemometrics and QSAR research Group.
Visit our website at http://michem.unimib.it/


--------------------------------------------
HELP FILES
--------------------------------------------
Help is provided in the wsp_reduction routine. Further info on the algorithm can be found in the following paper:
D. Ballabio, V. Consonni, A. Mauri, M. Claeys-Bruno, M. Sergent, R. Todeschini
A novel variable reduction method adapted from space-filling designs 
Chemometrics and Intelligent Laboratory Systems (2014), 136, 147-154
Four datasets (simulated, aphid, coffee and biodeg) are provided as example files.
